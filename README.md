# Packemon example

An example package of [Packemon](https://github.com/milesj/packemon).

- [Gotta pack 'em all! | Packemon](https://packemon.dev/)

## Installation

    npm install @azu/packemon-demo

## Usage

```
npm install
npm run build
npm test
```
